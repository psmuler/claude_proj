俳句アプリ API テストツール

API設定

※ Google Apps ScriptでWebアプリとしてデプロイした後のURLを入力してください

俳句API テスト

俳句一覧取得

GET /api/haikus

地図用俳句データ取得

GET /api/haikus/map

俳句検索

GET /api/haikus/search

特定俳句取得

GET /api/haikus/{id}

詠み人API テスト

詠み人一覧取得

GET /api/poets

時代区分一覧取得

GET /api/poets/periods

詠み人検索

GET /api/poets/search

特定詠み人取得

GET /api/poets/{id}

詠み人の俳句一覧

GET /api/poets/{id}/haikus
